# Absenteeism Business Case

## Project Overview

This project develops a Machine Learning solution to predict excessive employee absenteeism using historical attendance and employee-related data. The objective is to help organizations identify absenteeism patterns, improve workforce planning, and support data-driven HR decision-making.

A Logistic Regression model was built after extensive data preprocessing, feature engineering, custom feature scaling, and statistical analysis. The final model and preprocessing pipeline were serialized for deployment and future predictions.

## Dataset Description

The dataset contains employee information such as:

* Reasons for Absence
* Transportation Expense
* Age
* Body Mass Index (BMI)
* Education
* Children
* Pets
* Workload Metrics
* Absenteeism Hours

The target variable is generated by classifying employees as:

* **0 → Normal Absenteeism**
* **1 → Excessive Absenteeism**

based on whether absenteeism hours exceed the dataset median.

---

## Data Preprocessing

### Feature Selection

The following variables were removed:

* Absenteeism Time in Hours
* Day
* Daily Work Load Average
* Distance to Work

### Custom Scaling

A custom scaler was developed using Scikit-learn's:

* BaseEstimator
* TransformerMixin
* StandardScaler

Certain categorical features were excluded from scaling:

```text
Reason_1
Reason_2
Reason_3
Reason_4
Education
```

All remaining numerical features were standardized before model training.

## Model Interpretation

Feature importance was analyzed using:

### Coefficients

Measures the impact of each feature on absenteeism likelihood.

### Odds Ratios

```python
summary['Odds Ratio'] = np.exp(summary['Coefficients'])
```

Odds Ratios were used to identify factors that increase or decrease the probability of excessive absenteeism.
