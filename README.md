## 📈 Stock Price Predictor

**Folder:** `Stock-Price-Predictor/`

A machine learning system that predicts stock closing prices using historical market data and regression-based models.

### 🔹 Overview

This project focuses on analyzing historical stock market data and building predictive models to forecast future prices.

### ⚙️ Features

* Historical stock data collection using Yahoo Finance
* Data preprocessing and feature engineering
* Feature scaling using StandardScaler
* Time-series visualization and trend analysis
* Linear Regression model implementation
* Random Forest Regression model
* Ensemble learning approach
* Model evaluation using regression metrics
* Actual vs Predicted visualization
* Error trend analysis

### 🤖 Models Used

* Linear Regression
* Random Forest Regressor
* Ensemble Regression Approach

### 📊 Performance

* Strong regression stability across datasets
* MAE ~ 3 (varies by stock and timeframe)
* Improved robustness using ensemble methods

## 📉 Customer Churn Prediction

**Folder:** `Customer-Churn-Prediction/`

A machine learning project focused on analyzing customer behavior to predict churn (customer attrition).

### 🔹 Overview

This project aims to identify patterns in customer behavior that indicate whether a customer is likely to discontinue a service.

### 📊 Current Progress

* Data cleaning and preprocessing completed
* Missing values handled
* Categorical encoding performed
* Exploratory Data Analysis (EDA) completed
* Feature distributions analyzed
* Correlation analysis between variables performed
* Initial business insights derived

### 🧠 Next Steps

### 🤖 Planned Models

### 📈 Future Evaluation Metrics
