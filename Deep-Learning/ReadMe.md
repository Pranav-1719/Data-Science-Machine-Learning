# Deep Learning Practice

This repository documents my deep learning journey through hands-on implementations, experiments, datasets, and projects. It serves as a collection of notebooks and practical applications focused on understanding neural networks and modern deep learning techniques using TensorFlow, Keras, and related tools.

## Overview

The goal of this repository is to learn deep learning concepts through implementation rather than theory alone. Each notebook and project focuses on building intuition, understanding workflows, and developing practical machine learning skills.

## Current Contents

### Audiobooks Business Case

A complete deep learning workflow based on an audiobooks customer dataset.

Topics covered:

* Data preprocessing
* Feature scaling and balancing
* Dataset splitting
* Saving datasets in NPZ format
* Preparing data for neural network training

Files:

* `Audiobooks_data.csv`
* `Audiobooks_data_train.npz`
* `Audiobooks_data_validation.npz`
* `Audiobooks_data_test.npz`
* `Audiobooks_Preprocessing.ipynb`
* `Audiobooks_Machine_Learning.ipynb`

### MNIST Problem

Implementation and experimentation using the MNIST handwritten digit recognition dataset.

Topics covered:

* Data loading and preprocessing
* Neural network creation
* Model training and evaluation
* Classification of handwritten digits

Files:

* `MNIST_Problem.ipynb`

### Practice Notebooks

General notebooks used for learning TensorFlow and deep learning fundamentals.

Files:

* `demo.ipynb`
* `tensorflow_practice1.ipynb`
* `TF_intro.npz`
